# QuickTasks Backend

This directory contains the Node.js and Express backend for the QuickTasks application.

## Setup and Running

1.  **Navigate to the backend directory:**
    ```bash
    cd backend
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the server:**
    ```bash
    npm start
    ```

The server will start on `http://localhost:3001`.
